# Activity 1 - Library Book Management System

## Purpose
Develop a basic *Java Library Book Management System* to track student book borrowing and practice:
- Arrays for data storage  
- Loops and conditionals for logic  
- Methods for modular coding

## Files
- LibrarySystem.java â€“ Main Java program implementing the system.

## Features
- *Main Menu*: Add Book | Update Book Status | Show All Books | Generate Report | Exit  
- *Add Book* (addBook()): Register new books (max 5), default status "Available"  
- *Update Book Status* (updateBookStatus()): Toggle status between "Available" and "Borrowed"  
- *Display All Books* (showBooks()): List all books with status  
- *Generate Report* (generateReport()): Show total, available, and borrowed books  
- *Handle Invalid Input*: Validates menu selections and book numbers  
- *Exit Program*: Graceful termination

## How to Run
1. Compile: javac LibrarySystem.java  
2. Run: java LibrarySystem  
3. Follow menu prompts to manage books and generate reports.

# Java Programming Laboratory Activities and Laboratory Exam

## Overview
This repository contains **Laboratory Activities 1 to 7** for the Java Programming course.  
Each activity demonstrates the application of **Object-Oriented Programming (OOP)** concepts and Java fundamentals, progressing from basic class design to advanced topics such as abstraction, encapsulation, and polymorphism.

---

## Table of Contents

### Activity 1 – Library Book Management System
- Arrays for data storage  
- Loops and conditional statements  
- Methods for modular programming  

### Activity 2 – Book List Management System
- Classes and objects  
- Constructors and methods  
- Arrays of objects  

### Activity 3 & 4 – Modeling a School Management System
- Inheritance using `extends`  
- IS-A and HAS-A relationships  
- Aggregation between classes  

### Activity 5 – Coffee Shop Rewards System
- Default and parameterized constructors  
- Static variables and methods  
- Real-world problem modeling  

### Activity 6 – Smart Home Control System
- Method overriding  
- Runtime polymorphism  
- Encapsulation and parent-type references  

### Activity 7 – Abstraction and Encapsulation in Java
- Abstract classes and interfaces  
- Enums and package organization  
- ArrayList and Iterator usage

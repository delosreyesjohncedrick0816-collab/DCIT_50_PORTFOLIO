Activity 2 – Book List Management System
Purpose

Develop a basic Java Book List Management System to manage book information and practice:

Classes and objects

Constructors and methods

Arrays for data storage

Loops and conditionals for logic

Basic Object-Oriented Programming (OOP)

Files

MyBookList.java – Main Java program that creates and manages book objects

book.java – Java class that defines book attributes and methods

Features

Create Books: Initialize books with title, author, year published, and current page

Display Book Details (display()): Show book information using a method

Change Page (changepage()): Update the current page of a selected book

Change Author (changeauthor()): Modify the author of a book

Book Collection: Store multiple books in an array

Filter Books: Display titles of books published after a specific year

Program Execution: Runs sequentially and displays output in the console
